# BuyTicket_ZhuHaiHongKong

Open the url in Chrome;

Press F12;

Select Network;

Select All;

Select the date of ticket you wanna buy;

Right click 'query.book.info.data';

Select Copy;

Select Copy as cURL;

Transfer the data into python readable type, and name it as 'content';

Run the python file.
